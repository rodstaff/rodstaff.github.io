Redux-React-Routes

"Redux-React-Routes" is coded in ReactJS + React-Router and using the Redux framework.   This app also proposes to greatly simplify ReactJS coding and Heroku deployment by using client side code in development but using the webpack-dev-server to do the (auto-update) server rendering in the background.   In other words, the only Express coding you will do (and it's only 5 lines of code!) is when you deploy to Heroku in production.   That is, there were two apps used here:   (1) Development  to create the bundle.js file, and (2) Production using a very simple server side rendering code in Express to serve the bundled file for Heroku deployment.   Please take a look especially at the "README.md" files in Github for more details.

myCV

"myCV" is my own resume and I coded it in ReactJS + React-Router w/o Redux. If you are a hiring manager or recruiter, please let me know if you would like to see it and I would be very happy to send you a link and a copy of the source code.

MyTextingApp

"MyTextingApp" is coded using the NodeJS+Express framework with Twilio SMS or text messaging utilizing Twilio RESTful API.   The app sends me a text after the visitor inputs precisely the correct phone number format.   I created this from scratch.

Panacea

"Panacea" is adapted from a tutorial and is coded in Ruby on Rails using SendGrid and MailerChimp plugins for email and subscription purposes respectively.   It is an app where users share their personal anecdotes to make other people laugh or smile.   This is an experimental site.

Home

I created this website using a static site generator called Jekyll (based on Ruby) which I became comfortable using in less than an hour!   I chose Jekyll for its simplicity and for easy editing remotely on your local terminal.   All the source code of this portfolio page is available on Github.
