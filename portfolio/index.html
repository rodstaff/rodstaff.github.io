Some of the Websites/Apps that I have created.   Enjoy them as much as I enjoyed creating them freely employing some artistic liberties.

Note:   Except for this website, all websites/apps that are listed here are deployed in www.heroku.com.   There is an initial load time.
ProSeed

"Pro-Seed" is a demo website which I coded from scratch in ReactJS in ES6/NodeJS/Express using Bootstrap and in-line styling.   It's a scroll type website.   If you like this and need help upgrading your website using ReactJS, let me know.

ProSeed2

"Pro-Seed2" is an exact replica of "Pro-Seed" above, differing only in the memory space taken by the Team section photos.   (Data sizes are indicated at the top/bottom part of the images/photos, fyi.)   This is intentional to show how high-resolution photos can affect loading times.   ReactJS can make your website or app snappy and quick, but you still have to optimize all photo/image resolution.

JustDoIt

"Just-Do-It" is coded in ReactJS in ES6/NodeJS/Express.   It is a todo list using CRUD functions.   A seemingly simple app that requires lengthy and tricky code!

Panacea

"Panacea" is coded in Ruby on Rails using SendGrid and MailerChimp plugins for email and subscription purposes respectively.   It is an app where users share their personal anecdotes to make other people laugh or smile.   This is an experimental site.

MyWebsite

I created this website using a static site generator called Jekyll which is based on Ruby.   I chose Jekyll for its simplicity and for easy editing and uploading remotely on your local terminal.   All the source code of this portfolio page is available on Github.
