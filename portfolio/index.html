Some of the basic Apps I have done.

Note: The links might take a few seconds to load.

Panacea

"Panacea" is coded in Ruby on Rails using SendGrid and MailerChimp plugins. It is a newsletter app where prospective users visit the site to share their personal funny stories to make other people laugh or smile. This is an experimental site.


Redux-React-Routes

"Redux-React-Routes" is coded in ReactJS using the Redux framework. It is a simple app that I combined using at least three sources and it serves as template for much bigger and complex apps that would benefit from the Redux framework. It is a playful presentation of some famous people and the Mutant Ninja Turtles!


MyTextingApp

"MyTextingApp" is coded in JavaScript using NodeJS/Express with Twilio SMS texting. It is a very simple app that sends the author a text after the visitor inputs the correct phone number format.


Home

I created this website using a static site generator called Jekyll and posted at Github.
