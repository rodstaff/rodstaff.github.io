Some of the Apps that I have created.

Note: The links might take a few seconds to load.

Panacea

"Panacea" is adapted from a tutorial and is coded in Ruby on Rails using SendGrid and MailerChimp plugins. It is a newsletter app where prospective users visit the site to share their personal funny stories to make other people laugh or smile. This is an experimental site.


Redux-React-Routes

"Redux-React-Routes" is coded in ReactJS using the Redux framework. This app is not as simple as it appears: I used and combined at least three sources to create the beginnings of template for much bigger and complex apps that would benefit from the Redux framework. It is a playful presentation of some famous people and the Mutant Ninja Turtles!


MyTextingApp

"MyTextingApp" is coded in JavaScript using NodeJS/Express with Twilio SMS texting. I created this app from scratch and it sends me a text after the visitor inputs the correct phone number format.


Home

I created this website using a static site generator called Jekyll. All the source code of this portfolio page is available on Github.
